Droits d'accès

Modules auxquels l'utilisateur peut accéder.
Si un module A nécessite l'accès au module B pour certaines opérations, les accès au module B seront autorisés, dans la limite de ce qui est strictement nécessaire, par les autorisations du module A.
Par exemple: Le module gestion de projet donne autorise l'accès au prix inscrit dans le module gestion des offres.
